# Phase 3 — 건물현황및냉방부하현황 시트 (`건물현황및냉방부하현황`)

> 시트명 주의: '건물개요' 아님. 정확히 `건물현황및냉방부하현황` 사용.

## 열 구성
```
A=항목 | B=기재값 | C=단위 | D=검증값(수식) | E=판정(수식)
```

---

## 블록 구성

> **행 배치·항목명은 아래 시트4 셀 맵(행3~27)을 그대로 사용한다.**
> 아래는 내용 설명이며, 셀 위치·라벨은 템플릿이 우선한다.

### [건물 현황 블록]
기본 건물 정보 항목:
- 건물명
- 소재지
- 건물용도
- 연면적(㎡)
- 연면적(평) → `=연면적㎡/3.30579` 수식
- 냉난방면적(㎡)
- 냉난방면적(평) → `=냉난방면적㎡/3.30579` 수식
- 냉난방방식
- 건축연도

### [냉방부하 블록]
- 냉방 시간최대부하(kcal/h 또는 kW)
- 냉방 시간최대부하(kW 환산) → 수식
- 냉방 시간최대부하(RT 환산) → 수식
- 냉방 최대부하일 부하량(kcal/D 또는 RTh)
- 냉방 최대부하일 부하량(RTh 환산) → 수식
- E열 판정: 부하계산서 시트(시트3)와 일치 여부

### [난방부하 블록] (냉온열겸용인 경우)
- 난방 시간최대부하 (kW·RT 환산)
- 난방 최대부하일 부하량 (RTh 환산)
- 난방 총부하 ≤ 냉방 총부하 판정

---

## 판정 수식 예시
```python
# 냉방부하 일치 여부 (시트3 값과 비교)
ws["E{행}"] = f'=IF(ABS(D{행}-건물부하계산서_기준값)<건물부하계산서_기준값*0.01,"✅","❌")'
```

## 시트4 셀 맵 (`건물현황및냉방부하현황`)


행2 헤더: A2=`항목` B2=`기재값` C2=`단위` D2=`검증값(수식)` E2=`판정`
행 배치 고정:

| 행 | A열 |
|----|-----|
| 3 | `[건물 현황]` |
| 4 | 건물명 |
| 5 | 소재지 |
| 6 | 건물용도 (B열: `{코드} {명칭}` 예 `01 업무`; C열 빈칸) |
| 7 | 건물규모 (B열: `지상 {n}층 / 지하 {n}층`) |
| 8 | 연면적(㎡) → D8=`=B8/3.30579` |
| 9 | 연면적(평, 기재) → E9: D8과 ±1평 비교 수식 `=IF(ABS(D8-B9)<1,"✅","⚠️")` |
| 10 | 냉난방면적(㎡) → D10=`=B10/3.30579` |
| 11 | 냉난방면적(평, 기재) → E11: `=IF(ABS(D10-B11)<1,"✅","⚠️")` |
| 12 | 냉난방방식 |
| 13 | 건축연도 |
| 14 | `[냉방부하]` |
| 15 | 냉방 시간최대부하(기재) (B=숫자, C=단위) |
| 16 | 냉방 시간최대부하(RT 환산) → C16=`RT`, D16=환산 수식, E16=`=IF(ABS(D16-'건물부하계산서'!B22)<'건물부하계산서'!B22*0.01,"✅","❌")` (부하계산서 `무`면 E16=`해당없음`) |
| 17 | 냉방 최대부하일 부하량(기재) |
| 18 | 냉방 최대부하일 부하량(RTh 환산) → C18=`RTh`, D18=환산 수식, E18=`해당없음` (일부하량은 부하계산서에서 다루지 않음) |
| 19 | 심야 시간최대부하 (B=`0`, C=`kcal/h`) |
| 20 | 심야 최대부하일 부하량 (B=`0`, C=`kcal/D`) |
| 21 | 냉방시간 (B열: `{HH}시~{HH}시 ({N}h)` — 두 자리, 예 `08시~18시 (10h)`) |
| 22 | `[난방부하]` |
| 23 | 난방 시간최대부하(기재) |
| 24 | 난방 시간최대부하(RT 환산) |
| 25 | 난방 최대부하일 부하량(기재) |
| 26 | 난방 최대부하일 부하량(RTh 환산) |
| 27 | 난방 총부하 ≤ 냉방 총부하 → E27 판정 |

- 냉열전용이면 행23~26 B열 `해당없음`, 행27 B열 `해당없음`·E열 `✅`
- 행19·20 (심야부하): 기재 없으면 B열 `0` (숫자)
