> 판단이 갈릴 수 있는 모든 지점을 이 파일이 고정한다. **여기 없는 임의 판단 금지.**
> 문서 판독이 애매한 값은 "가장 명시적인 표(요약표 > 데이터시트 > 서술문)" 순으로 채택하고,
> 상충하면 요약표 값을 기재값으로 쓰고 교차검증 카탈로그(아래)에 올린다.

## 2. 펌프 담당 냉동능력(RT) 정의 — LPM/RT 분모

**펌프는 용도가 여러 종류일 수 있으므로 먼저 구분한다.** `펌프` 시트 `s+1` 행에
`{설비명} ({용도 구분} / {순환유체})` 형식으로 적는다.

| 용도 구분 | 순환유체 | 담당 RT (고정) | D열(비고) 문구 |
|-----------|----------|---------------|----------------|
| **열원펌프** | 냉각수 | 담당 냉각탑 **공칭 CRT** / 펌프 1대 담당 계통 | `냉각탑 공칭 CRT 기준` |
| **축열펌프** | 브라인 | 열원기기 **제빙능력 RT** / 펌프 1대 담당 계통 | `제빙능력 RT 기준` |
| **방열펌프** | 브라인 | 담당 회로 열교환기 정격 RT(방냉 기준) / 펌프 1대 담당 계통 | `열교환기 정격 RT 기준` |
| **축방열펌프**(축열·방열 겸용) | 브라인 | 담당 회로 열교환기 정격 RT(방냉 기준) / 펌프 1대 담당 계통 | `열교환기 정격 RT 기준` |
| **부하펌프**(2차측·냉수) | 냉수 | 담당 회로 열교환기(또는 냉동기 냉수측) 정격 RT / 펌프 1대 담당 계통 | `냉수 회로 정격 RT 기준` |

- 겸용 펌프는 **방열 기준**으로 담당 RT를 잡는다(방열이 주간 피크를 감당하므로).
- 위에 없는 용도(보충수·응축수 등)는 순환펌프가 아니므로 `펌프` 시트에 넣지 않는다.

- 다른 분모(냉동기 냉방 RT 등)로의 참고 계산 행 **추가 금지**.
- 판정이 이탈이면 체크리스트 행27 = ⚠️, 업체확인필요사항 카탈로그 P2로 기재.

## 3. 축열조 축열능력 표준 계산법

### 3.1 물성치 고정값 (조회 금지, 이 값 그대로)

| 매체 | ρ (kg/㎥) | Cp (kcal/kg℃) | L (kcal/kg) |
|------|-----------|----------------|--------------|
| 물(액체) | 1000 | 1.00 | — |
| 얼음 | 917 | 0.50 | 79.68 |
| 에틸렌글리콜 25% | 1036 | 0.90 | — |
| 에틸렌글리콜 30% | 1040 | 0.82 | — |
| 프로필렌글리콜 25% | 1020 | 0.90 | — |
| 프로필렌글리콜 30% | 1025 | 0.85 | — |

### 3.2 빙축열(관외착빙형·완전동결형 등 조 내 코일형) — 표준 산식

1. 축열매체(물/얼음) 체적 V_ice_zone = 내부 가로 × 세로 × **축열매체높이**(기재 시; 없으면 내부높이) − 코일·헤더 체적(기재 시)
2. 잠열 = V_ice_zone × IPF × 917 × 79.68 [kcal]
3. 순환매체(브라인) 체적 = 브라인 보유량 기재값(데이터시트 우선), 없으면 내부체적 − V_ice_zone
4. 순환매체 현열 = V_브라인 × ρ × Cp × |주간 순환매체 입출구 ΔT| [kcal]
5. **축열매체(물) 현열은 0으로 고정** — phase5 검증 블록의 축열매체 ΔT 셀(B42)에 항상 0 입력 (온도범위가 명시된 경우에도 표준 산식에서 제외)
6. 총 계산 축열능력 = (2 + 4) ÷ 3024 [RTh] → 설계값과 차이율 비교 (<10% ✅)

### 3.3 캡슐형(아이스볼 등)

잠열 = 캡슐 총체적 × IPF × 917 × 79.68. 순환매체 현열은 3.2와 동일.

### 3.4 수축열

현열 = 내부체적 × 1000 × 1.0 × |입출구 ΔT| ÷ 3024 [RTh].

## 4. 시간대별 운전계획 수치 채택

- 계획서의 시간대별 표를 **원문 단위 그대로** 채택한다(RT/USRt/kW/Kcal/hr). **환산 금지** — v14 규칙.
- 축열조 누계의 기준(0점)은 **축열 시작 직전**. 24시간 표를 축열 시작 시각이 첫 행이 되도록 회전시킨다.
- 시간대별 표가 **없으면 만들지 않는다**. 평균축냉량·평균방냉량으로 균등 배분해 표를 지어내지 말고,
  시트에 미수록으로 기록한 뒤 업체확인 카탈로그 K5로 기재한다.
- 냉방시간 외 시간대 부하 = 0.
- 열 정의(누계형 축열량 열, 총출력 축열운전 열 등) 판별은 `phase4_operation_plan.md`의 함정 표를 따른다.

## 5. 업체확인필요사항 교차검증 카탈로그 (순서·문구 완전 고정)

### 5.0 트리거 규칙 — 결정론(반올림/유의성 판단 금지)

> 각 카탈로그 항목은 아래 트리거가 성립하면 **무조건** 기재한다. "차이가 반올림 수준이라
> 무시" 같은 판단을 하지 않는다(§7의 값 채택 규칙은 '기재값을 무엇으로 쓰나'의 문제이고,
> 카탈로그 기재 여부와 무관하다).
>
> 트리거 표의 괄호 안 숫자는 **판별 예시**(A빌딩 문서에서 실제로 성립했던 값)이며,
> 기준값이 아니다. 문서마다 성립하는 코드가 다르므로 **문서별로 트리거를 하나씩 판정**한다.

| 코드 | 트리거 (성립 시 무조건 기재) |
|------|------------------------------|
| K1 | 부하계산서에 계산방법(프로그램/기준)이 문자로 명시돼 있지 않으면 |
| K2 | 연면적이 두 곳에서 다르고 **차이 > 0.1%** 이면 (`43,202.00` vs `43,202.77` = 0.002% → **기재 안 함**) |
| K3 | 축열률이 두 값으로 병기(고객제시식 vs 설계식)돼 값이 다르면 (`47.0` ≠ `50.2` → 기재) |

> 확실할 때만' 단서는 전용 시트가 생기면서 불필요해졌다). 값을 읽을 수 없으면 `확인불가`로 두고
> 추정값을 넣지 않는다. 그 외 코드는 트리거 성립 여부만 보고 기계적으로 판정한다.

- **시트12 `업체확인필요사항`은 양식만 만들고 비워 둔다(v17).** 사용자가 직접 작성한다.
  분석이 찾아낸 항목은 아래 두 곳이 전담한다:
  - `체크리스트` 행7~27 — 21개 심사항목의 ⚠️/❌ 판정
  - `체크리스트` 행28~ — **카탈로그 발견 항목 전부**(아래 표 순서). A열=`추가`, B열=카탈로그
    표준 제목, C열=카탈로그 D형식, D열=카탈로그 E형식, E열=`⚠️`, F열=빈칸
- **출처 라벨은 3개만 사용**: `설치내역서`(본문 요약표), `설계서`(부하계산서·빙축열 시스템 설계서),

## 5.5 상호검증 허용오차 (v17) — 상대오차 고정

`상호검증` 시트와 카탈로그 트리거는 아래 허용기준을 **그대로** 쓴다. 기준 안이면 **일치**로 보고
카탈로그에 올리지 않는다.

| 대상 | 허용기준 | 근거 |
|------|----------|------|
| **연면적** | **0.1%** | 건축물대장·설계서 간 끝자리 처리 차이 |
| **심야전력 설비용량(입력합계)** | **0.1%** | 기기별 입력 반올림 누적 |
| **유량·열량**(열교환기·링블로워·냉각탑·순환수량·축열량 등) | **1%** | 단위 환산·기기 선정 여유 |
| 그 외(수량·회전수·온도·능력·축열률·입력 kW 등) | **완전일치** | 설계값이므로 달라질 이유가 없다 |

- 시트에서는 `허용기준` 열에 문구(`0.1%`·`1% (유량)`·`완전일치`)를, `허용오차(수식)` 열에
  `=C{r}*0.001` 형태의 **수식**을 넣어 값이 바뀌어도 자동 반영되게 한다.
- **내부체적 vs 총유체량처럼 애초에 다른 값**은 대조 대상에 넣지 않는다(오탐 방지).
  대조표 아래 ※ 행에 그 사실을 남긴다.

## 6. 표기 사전 (전 시트 공통, 글자 그대로)

| 대상 | 고정 표기 |
|------|-----------|
| 시간 라벨 | 항상 두 자리: `22:00~23:00`, `23:00~24:00`, `00:00~01:00` … (`0:00`, `8:00` 금지) |
| 시간 범위+시간수 | `22:00~08:00 (10h)` 형식 |
| 법인 표기 | `(주)`로 정규화 (`㈜` 금지) — 예: `○○타워 ((주)○○○)` |
| 계약종별 코드 | 괄호 안 로마숫자 붙여쓰기: `심야전력(을II)`, `일반용 전력(을)` (`(을 II)` 공백 금지) |
| 건물용도 | `{코드} {명칭}` 형식: `01 업무` (`업무(01)` 금지) |
| 시간 라벨(운전 외) | 시각은 두 자리+`시`: `08시~18시 (10h)` (`8시` 금지) |
| 코드+명칭 | 원문에 코드가 있으면 `{코드}.{명칭}` (예: `02.빙축열 / 04.관외착빙형`) |
| 숫자 천단위 | 능력·용량·금액 등 물리량은 천단위 쉼표. 단, **치수(mm)·모델 규격의 정수**는 쉼표 없이(`L2078 × W766 × H1800`, 중량 `제품 3400 / 운전 3755`) |
| 재질/구성 병기 | 콜론+슬래시: `Casing: 아연도금강판 / 축열코일: P.E` |
| 치수 | `{가로} × {세로} × {높이} m ({체적} ㎥/조)` — 숫자 사이에만 ` × `, 단위 m은 마지막 1회 |
| 수전설비 | `수전전압 {n} V / 용량 {n} kVA` |
| 재질(두께) | `{재질}({두께})` 붙여쓰기 — 예: `EPS판넬(100T)`, `EVA(1.5T)` |
| 원문 그대로 원칙 | 문서의 단위·표기(50T, CMM 등)를 임의 변환하지 않는다 (단, 위 고정 표기 항목은 예외) |
| RT→kW(h) 수식 | `={RT값}*3.517` — 정수면 소수점 없이(`900`), 소수면 그대로(`406.5`). **적용 범위: 시트5 열원기기 비교 블록 등 환산이 필요한 곳만. 시트6·7 시간대별 운전계획은 단위 무변환(환산 금지)** |
| IPF | % 숫자로 기입 (95, 0.95 금지). 소수식이 필요한 수식에는 `/100` 포함 |
| 빙축열 여부(열원기기 B33) | `예(제빙능력표 포함)` 또는 `아니오` |
| 열교환기 형식(B6) | `판형` (부가 표기 금지) |
| 회전수 | `{정수}rpm / {정수}P` — **rpm 숫자는 쉼표 없이** (`1750rpm / 4P`) |
| 재질(두께) 붙여쓰기 | 공백 없이: `우레탄판넬(50T)`, `EPS판넬(100T)` |
| 펌프 설비명 | 설치내역서 표기 그대로, **접미사 `_1`·`_2` 등을 붙이지 않는다** (`축방열펌프`, `열원펌프`) |
| 축열형식(축열조 B5) | `{대분류코드}.{명칭} / {중분류코드}.{명칭}` (예: `02.빙축열 / 04.관외착빙형`) |
| 참조 문구 `vs 기재` | 숫자 천단위 쉼표 포함: `vs 기재 {n} RTh` |
| 냉난방방식(건물현황 B12) | 축열형식 기반: `빙축열식(간접열교환)` (미기재 처리 금지 — 축냉설비이므로 항상 성립) |
| 계약번호(고객현황 B13) | 설치내역서 표지·계약 페이지의 전자계약번호를 반드시 전사 (놓치면 안 됨) |
